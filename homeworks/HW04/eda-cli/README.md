# HW04: EDA CLI & HTTP Service

Проект для разведочного анализа данных (EDA) и проверки качества датасетов.
Включает в себя консольную утилиту (CLI) и HTTP-сервис (REST API) на базе FastAPI.

## Требования

- Python 3.11+
- [uv](https://docs.astral.sh/uv/) установлен в систему

## Инициализация проекта
